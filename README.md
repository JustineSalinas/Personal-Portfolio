# Personal Portfolio

A modern, sleek personal portfolio built with Next.js, Tailwind CSS, and TypeScript. It features a stunning Bento grid design to showcase your projects, skills, and bio.

## Features

- **Next.js App Router**: Lightning-fast, modern routing.
- **Tailwind CSS**: Utility-first styling for rapid and beautiful UI development.
- **TypeScript**: Strictly typed for a robust developer experience.
- **Bento Grid Layout**: A responsive, visually striking layout for your content.
- **Centralized Content Structure**: Easily manage all your portfolio data (projects, skills, bio) from a single file (`src/data/index.ts`).

## Getting Started

First, install the dependencies (make sure you do this before starting the development server):

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

- `src/app/`: Next.js standard App Router directory.
- `src/components/`: Reusable tailored components safely organized by UI parts and broader page sections.
- `src/data/`: Central hub for customizing data shown on your portfolio.
- `public/`: Place your images, SVGs, and other static assets here.
